## 💬 QUICK ANSWERS

**Q: How do I start the app?**  
A: Run `npm run dev` and open http://localhost:5173/

**Q: How do I connect to the backend?**  
A: Backend automatically proxies to http://localhost:3000. Make sure it's running!

**Q: How do I create a component?**  
A: Create a new JSX file in `src/components/` and export it

**Q: How do I add a custom hook?**  
A: Create a new JS file in `src/hooks/` starting with `use`

**Q: How do I style with Tailwind?**  
A: Add Tailwind classes directly to JSX elements: `<div className="bg-blue-600 text-white">`

**Q: How do I deploy to production?**  
A: Run `npm run build` and deploy the `dist/` folder
